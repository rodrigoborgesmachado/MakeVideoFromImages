# Make Video From Images

A desktop application built with .NET to automatically generate videos from large collections of photos.

  <img src="https://www.sunsalesystem.com.br/img/make_video.gif" alt="Make Video From Images Demo" width="900">

## Overview

Make Video From Images was created to solve a real-world challenge: generating a dynamic video from thousands of photos distributed across multiple folders while maintaining a balanced distribution and avoiding premature repetitions.

Instead of manually creating slideshows or editing timelines, the application automatically organizes images and generates a complete video presentation.

## Features

* Support for multiple image folders
* Balanced distribution between photo collections
* Smart randomization algorithm
* Prevents image repetition until all photos have been displayed
* Automated video generation
* Simple and intuitive desktop interface
* Optimized for large photo collections

## Technologies

* C#
* .NET
* Windows Desktop
* Image Processing
* Automated Video Generation

## Use Cases

* Wedding retrospectives
* Family photo slideshows
* Event presentations
* Digital photo frames
* Corporate presentations

## Results

The application significantly reduces the manual effort required to create professional-looking video presentations from thousands of images, making it ideal for events, celebrations, and multimedia showcases.
